# Lista de Compras

Aplicação prática de uma lista de compras interativa desenvolvida com o objetivo de gerenciar itens de forma simples. O sistema permite a inserção de produtos ou materiais, marcação de itens concluídos e a remoção de elementos da lista, auxiliando na organização diária.

## Funcionalidades

*   **Adição de Itens:** Campo de entrada de texto estruturado para registrar novos produtos ou tarefas a serem lembradas.
*   **Check-out Dinâmico:** Opção de marcação visual para sinalizar quais itens já foram comprados ou tarefas concluídas.
*   **Remoção Individual:** Botão dedicado para excluir itens específicos da lista de maneira rápida.

## Tecnologias Utilizadas

*   **Front-end:** HTML5, CSS3 e JavaScript (ES6+)
